# Shortener – собственный сервис коротких ссылок с QR-кодом

Лёгкий, без-авторизационный, REST-ориентированный проект на **Yii2 + Bootstrap 5 + jQuery**.  
Сжимает длинные URL, отдаёт QR-код, считает переходы.

---

## 📁 Структура проекта
```
project/
├── config/              # конфигурация приложения
├── controllers/         # SiteController (создание) + RedirectController (редирект)
├── models/              # ShortUrl + Log
├── migrations/          # одна миграция создаёт 2 таблицы
├── views/               # единственная страница «сжать»
├── web/                 # корень сервера, assets, css, js
└── runtime/             # кеш, логи Yii
```

## 🚀 Установка и запуск
1. **Склонировать**
   ```bash
   git clone https://github.com/Evgunt/testQr.git
   ```

2. **Поставить зависимости**
   ```bash
   composer install
   ```

3. **Настроить БД**  
   Создайте БД и пропишите DSN в `config/db.php`.

4. **Накатить миграции**
   ```bash
   php yii migrate/up
   ```

5. **Web-сервер**
   - прод: настройте `web/` как document-root
   - dev: `php -S 127.0.0.1:8080 -t web`

6. **Открыть**
   На главной форма «Вставьте ссылку»  
   После сжатия получаете короткий вида `http://127.0.0.1:8080/a1b2c3d4` и QR-код.

---

## 📊 Как это работает
1. Пользователь вставляет URL
2. Сервер проверяет живость (curl), генерирует 8-символьный `hash`, сохраняет в `short_url`
3. Возвращает JSON: `{hash: "https://host/xxxyyzzz", qr: "https://api.qrserver.com/..."}`
4. При переходе по короткой ссылке `RedirectController`:
   - увеличивает `counter`
   - пишет запись в `log` (IP, время)
   - делает 302-редирект на `long_url`
